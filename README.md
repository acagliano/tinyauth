# TINYAUTH
TinyAuth is a service to allow people login to apps on their calculator.

# HOW TO USE
You simply create an account on [TinyAuth's website](https://tinyauth.cagstech.com/) and download the keyfile.

Then you put the keyfile on your calculator and...

Magic happens! You open a supported program (Ex. TI-TREK) and select that keyfile.

You get magically logged in!!

Enjoy online programs you're gonna use!!
